# Sistema de Gestión de Refugio de Animales

## Descripción

Este proyecto consiste en un menu de consola desarrollado en Java para gestionar un pequeño refugio de animales.

El sistema permite:
- registrar animales,
- mostrar animales registrados,
- buscar animales por ID,
- cambiar el estado de un animal,
- ejecutar sonidos según el tipo de animal.

---

# Temas aplicados

En el proyecto se utilizaron los siguientes temas:

- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- ArrayList
- Getters y setters
- Constructores
- Sobrescritura de métodos (@Override)
- Validaciones básicas
- Try/Catch

---

# Estructura del proyecto

El proyecto contiene las siguientes clases:

## Animal
Clase base del sistema.

Atributos:
- id
- nombre
- edad
- estado

Métodos:
- mostrarInfo()
- hacerSonido()

---

## Perro
Clase hija que hereda de Animal.

Se sobrescribe:
- hacerSonido()

---

## Gato
Clase hija que hereda de Animal.

Se sobrescribe:
- hacerSonido()

---

## Refugio
Clase encargada de administrar la colección de animales usando ArrayList.

Funciones principales:
- registrar animales
- mostrar animales
- buscar por ID
- cambiar estado
- ejecutar sonido

---

##  App
Clase principal que contiene el menú del sistema.
